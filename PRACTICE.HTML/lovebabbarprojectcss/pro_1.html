when an id is given to an html element it serves as a unique identifier for that element. on the otherhand if class is given to a html element it specifies that it belongs from that class . id is unique for a single element but more than one elements can have same class. we can add multiple classes to an element at the same time. we add css in a html file by three ways. 1> inline css 2> internal css 3> external css a css selector is used to select an html element for styling element selector is used to select an element based on tagname class selector is used to select an element with given class we can group selectors universal selector selects all the elements of the html file if we change anything in the universal selector then it can change styling of all the elements that are the part of the given html file. inline css override both the internal and external css
